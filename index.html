无 render 只有 template

Vue 会把在运行时收到的 template 编译成 render。 
                  
const Counter = {
  template: `(JSX形式的组件内容)`,
  setup() {
    const count = ref(0)
    const click = () => {
      count.value += 1
    }
    return {
      count,
      click,
    }
  }
}

手写 render

直接提供 render 这样编译模板的环节步没有了。与编译产生的 render 一样，都是通过 Vue API 直接创建 VNode 树。 
                  
const Counter = {
  setup() {
    const count = ref(0)
    const click = () => {
      count.value += 1
    }
    return (
      使用如下原子 VNode 操作创建的 VNode 树 
      createVNode('comp', { class: ... }, 'str'/[chilren])
      createElementVNode('elem', { class: ... }, 'str'/[children])  
    )
  }
}

手写 render 并借助语法糖 h 简化

使用语法糖 h 可以在一定程度上简化在 render 中 VNode 树的创建工作。 
                  
const Counter = {
  setup() {
    const count = ref(0)
    const click = () => {
      count.value += 1
    }
    return (
      使用如下语法糖 h 函数来创建的 VNode 树 
      h('elem/comp', { class: 'cls', ... }, 'text'/[chilren])  
    )
  }
}
